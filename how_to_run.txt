===========================================

Windows (CMD):
---------------

ðŸ”¹ Install required Python packages:

pip install -r requirements.txt

- Flask
- TensorFlow
- TensorFlow Hub
- Pillow
- NumPy


===========================================
Step 2: Run the Flask App
===========================================

Ubuntu/Linux:
---------------
python3 app.py

Windows:
---------------
python app.py


===========================================
Step 3: Access in Your Browser
===========================================

After running the app, open your browser and go to the link you see in your terminal after running python3 app.py this command.



You can now upload an image, choose an art style, and see:
- The predicted style of your image
- A new image with that style transferred onto it
